Appliance model

The appliance model provides the customer with several machines — one machine with KK-Auditor and KK-Manager pre-installed, and the remaining machines with KK-Cryptoservice pre-installed. Follow these steps to get Kripta Key up and running:

  1. Install CLIKK on a device the administrators will be using, following Installing CLIKK.

  2. Set up and configure KK-Auditor, following KK-Auditor setup and configuration.

  3. Set up and configure KK-Manager, following KK-Manager setup and configuration.

  4. Perform key ceremony, initializing the main KK-Manager instance and registering the KK-Auditor instance to it, following Key ceremony.

  5. Register each KK-Cryptoservice instance, following Registering KK-Cryptoservice.

Kripta Key is now ready to use.
