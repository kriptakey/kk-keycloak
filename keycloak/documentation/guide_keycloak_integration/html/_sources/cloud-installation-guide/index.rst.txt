Cloud Installation Guide
========================

Preface
-------

Intended reader
~~~~~~~~~~~~~~~

This document is intended for administrators as a guide to installing Kripta Key (KK-Cryptoservice, in particular)
on cloud with cloud-based applications.


Document scope
~~~~~~~~~~~~~~

This document explains the process of installing this version of Kripta Key on cloud for use
with cloud-based applications. This document features a step-by-step guide to installing this version
of Kripta Key on `Alibaba Cloud`__.

__ https://alibabacloud.com/
